# PostgreSQL Aggregate Functions

Aggregate functions in PostgreSQL perform calculations on a set of rows and return a single value. They are commonly used with the `GROUP BY` clause to summarize data in meaningful ways.

Imagine you own a **restaurant** and want to analyze customer orders:
- **How many orders were placed?**
- **What is the average order value?**
- **What was the most expensive dish ordered?**

PostgreSQL aggregate functions help answer these questions efficiently.
